# 邮件技能
## 用法
- 用法:`python scripts/send_email.py \
  --email <发件人邮箱> \
  --auth-code <授权码> \
  --to <收件人邮箱> \
  --subject <邮件主题> \
  --content <邮件内容>`
- 例子: `python scripts/send_email.py \
  --email "123456789@qq.com" \
  --auth-code "your_auth_code" \
  --to "123456789@qq.com" \
  --subject "邮件主题..." \
  --content "邮件内容..."`
## 执行技能
- 使用:execute_skill_script工具调用`scripts/send_email.py`脚本
